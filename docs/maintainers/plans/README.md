# Active maintainer plans

This directory is reserved for active or near-active Mini App planning/spec documents that still guide current work.

Completed or superseded dated plans, sign-off notes, QA checklists, handoff snapshots, and implementation-status logs should be moved to `../archive/dated-plans/`.

It is valid for this directory to contain either:
- only this index file, when no active plan needs to stay live here, or
- one or a small number of current plans/specs that are intentionally guiding the next Mini App work.
