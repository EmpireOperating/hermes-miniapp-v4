# Code of Conduct

## Our standards

We want this project to be a respectful, collaborative place for contributors and users.

Examples of positive behavior:
- being respectful and constructive
- assuming good faith while discussing technical disagreements
- giving and receiving feedback gracefully
- focusing on the project and the code, not on personal attacks
- helping make the project welcoming to people with different backgrounds and experience levels

Examples of unacceptable behavior:
- harassment, discrimination, or hateful conduct
- personal attacks, insults, or intimidation
- doxxing or sharing private information without permission
- disruptive trolling or bad-faith participation
- sexualized language or unwanted sexual attention

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, pull requests, or other contributions that violate this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainers privately through GitHub rather than escalating publicly.

## Scope

This code of conduct applies in project spaces, including issues, pull requests, discussions, and other official community channels tied to this repository.
